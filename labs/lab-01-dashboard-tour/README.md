# Lab 01: Dashboard Tour

## Overview

This lab guides you through exploring the OPCP dashboard interface. You will learn how to navigate the main sections, locate key information, and understand the layout of the management console.

## Prerequisites

Before starting this lab, ensure you have:

- **Software Requirements:**
  - A modern web browser (Chrome, Firefox, Edge, or Safari)
  - An active internet connection

- **Access Requirements:**
  - An OVHcloud account with OPCP service access
  - Valid login credentials for the OVHcloud Control Panel

- **Knowledge Requirements:**
  - No prior technical knowledge required
  - Familiarity with basic web browser navigation (clicking links, using menus)

## What You Will Learn

- How to log in to the OPCP dashboard
- How to identify the main navigation areas
- How to locate your service information
- How to find help and support resources

## Total Steps

This lab contains **8 steps** across 1 exercise.

## Estimated Duration

Approximately **15 minutes** to complete.

## How to Use This Lab

1. Run the setup script to verify your environment is ready:
   ```
   bash setup/setup.sh
   ```
2. Open the exercise file and follow the numbered steps in order.
3. Each step describes one action and what you should see as a result.
4. If you get stuck, refer to the solutions directory for guidance.

## Troubleshooting

- If the setup script reports an error, follow the suggested fix shown in the error message.
- If you cannot access the dashboard, verify your credentials are correct and your account is active.
- For additional help, contact your team lead or refer to the OVHcloud support documentation.
