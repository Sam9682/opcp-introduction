Canaux de support

Quand la documentation ne suffit pas, OVHcloud propose plusieurs moyens d'obtenir une aide directe de professionnels du support.

Système de tickets de support

Soumettez une demande de support via l'Espace Client OVHcloud. Les tickets sont suivis, priorisés selon leur gravité et traités par des équipes spécialisées. Idéal pour les problèmes non urgents, les questions techniques détaillées ou quand vous avez besoin d'un historique écrit de la résolution.

Ouvrir un ticket →

Support téléphonique

Parlez directement avec un agent de support pour les problèmes urgents ou quand vous avez besoin d'un accompagnement en temps réel. Le support téléphonique est disponible pour les clients avec des plans de support Premium ou Business. Consultez votre contrat pour les horaires de disponibilité et les numéros de contact.

Voir les niveaux de support →

Chat en direct

Disponible dans l'Espace Client OVHcloud pour les questions rapides et l'orientation. Les agents du chat peuvent vous aider avec les problèmes de compte, les questions de facturation et vous diriger vers la bonne documentation pour les questions techniques.

Démarrer le chat →

Conseils pour obtenir de l'aide efficacement

  • Consultez d'abord la documentation — De nombreuses questions trouvent déjà leur réponse dans les guides et tutoriels.
  • Soyez précis — Lorsque vous contactez le support, indiquez ce que vous faisiez, ce que vous attendiez et ce qui s'est réellement passé.
  • Incluez les identifiants — Fournissez votre identifiant de projet, le nom du service ou le numéro de ticket pour accélérer la résolution.
  • Vérifiez la page de statut — Avant d'ouvrir un ticket, vérifiez qu'il n'y a pas d'incident en cours affectant votre région.
  • Utilisez la communauté — D'autres utilisateurs ont peut-être déjà rencontré et résolu le même problème.
