Navigating the OPCP Interface

Beginner

Overview

Once you are logged in to the OVHcloud Manager, you need to know how to find your way around. Think of the interface like a well-organised office building: the main menu is the directory in the lobby, each section is a different floor, and you can always take the lift back to the ground floor (the dashboard).

This lesson covers the main menu sections, how to move between them, and how to return to the dashboard at any time.

The Main Menu

The left-side navigation menu is your primary way to access different parts of the OPCP interface. It is always visible on desktop screens and organises services into logical categories.

Compute Section

The Compute section is where you manage your processing power — the virtual machines and nodes that run your applications. This is like the engine room of your private cloud.

In this section you can view:

  • Active nodes and their current status
  • Resource usage (how much computing power is being used)
  • Options to add or modify compute resources

Storage Section

The Storage section manages where your data lives. Think of it as your filing cabinet — this is where files, databases, and backups are kept safe.

Here you will find:

  • Your CloudStore volumes and their capacity
  • Backup schedules and recovery options
  • Data usage statistics

Network Section

The Network section controls how your services connect to each other and to the outside world. Think of it as the road system connecting different buildings in a business park.

This section includes:

  • vRack private network configuration
  • Public IP address management
  • Firewall and security group settings
  • Bandwidth monitoring

Settings Section

The Settings section is where you manage your account preferences, billing information, and access controls. Think of it as the admin office where you handle paperwork and permissions.

Key items in Settings:

  • Account details and contact information
  • Billing and payment methods
  • User management (adding team members)
  • Security settings (passwords, two-factor authentication)
  • API credentials and access tokens

Moving Between Sections

Switching between sections is straightforward:

  1. Click any section name in the left menu to expand it
  2. Click a sub-item to open that specific page
  3. The currently active section is highlighted so you always know where you are

You can have only one section expanded at a time. Clicking a different section collapses the current one and expands the new selection.

Returning to the Dashboard

No matter where you are in the interface, you can always return to the main dashboard in two ways:

  1. Click the OVHcloud logo — Located in the top-left corner, clicking the logo always takes you back to the dashboard home
  2. Click "Dashboard" — The first item in the left menu is always a link back to the dashboard overview

Navigation Tips

  • Breadcrumbs — At the top of each page, a breadcrumb trail shows your current location (e.g., Dashboard > Compute > Nodes). Click any part of the trail to jump back to that level.
  • Search bar — Use the search bar at the top of the page to quickly find a specific service or setting by name.
  • Keyboard shortcut — Press the Home key or use your browser's back button to return to previous pages.
