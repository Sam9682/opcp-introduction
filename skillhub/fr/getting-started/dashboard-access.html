Accès au Tableau de Bord

Débutant

Présentation

L'Espace Client OVHcloud est votre panneau de contrôle central pour tous les services OPCP. Imaginez-le comme l'accueil d'un immeuble — c'est le point d'entrée unique où vous vous identifiez et êtes dirigé vers tout ce dont vous avez besoin.

Cette leçon vous montre comment accéder au tableau de bord, ce dont vous avez besoin pour vous connecter, et ce que vous verrez à votre arrivée.

URL du Point d'Accès

Le tableau de bord OPCP est accessible via l'Espace Client OVHcloud à l'adresse :

https://www.ovh.com/manager

Ajoutez cette adresse à vos favoris pour un accès rapide. C'est la même URL utilisée par tous les clients OVHcloud dans le monde — une fois connecté, vous ne verrez que vos propres services et données.

Prérequis de Connexion

Avant de pouvoir accéder au tableau de bord, assurez-vous d'avoir :

  • Un compte OVHcloud actif — Si vous n'en avez pas encore créé un, consultez la leçon Configuration du Compte
  • Votre e-mail de connexion — L'adresse e-mail utilisée lors de l'inscription
  • Votre mot de passe — Le mot de passe créé pour votre compte
  • Appareil d'authentification à deux facteurs (si activé) — Un téléphone ou une application d'authentification pour le code de sécurité

Étape 1 : Accéder à la page de connexion

Ouvrez votre navigateur et rendez-vous sur https://www.ovh.com/manager. Vous verrez l'écran de connexion OVHcloud avec les champs pour votre e-mail et votre mot de passe.

Étape 2 : Saisir vos identifiants

Tapez votre adresse e-mail enregistrée et votre mot de passe. Si vous avez activé l'authentification à deux facteurs, un code de vérification vous sera demandé après avoir saisi votre mot de passe.

Étape 3 : Arriver sur le tableau de bord

Après une connexion réussie, vous arrivez sur le tableau de bord de l'Espace Client OVHcloud. C'est votre base — une vue résumée montrant vos services actifs, l'activité récente et des raccourcis vers les actions courantes.

Ce que vous voyez sur le tableau de bord

La page d'accueil du tableau de bord est organisée en sections claires :

  • Cartes de résumé des services — Compteurs rapides de vos ressources actives de calcul, stockage et réseau
  • Activité récente — Une chronologie des modifications récentes, commandes et tickets de support
  • Actions rapides — Raccourcis vers les tâches courantes comme commander de nouveaux services ou consulter les factures
  • Menu de navigation — Le panneau latéral gauche qui vous dirige vers les catégories de services spécifiques (traité dans la leçon suivante)

Problèmes d'accès courants

Si vous ne parvenez pas à vous connecter, voici les raisons les plus courantes et leurs solutions :

  • Mot de passe oublié — Cliquez sur « Mot de passe oublié ? » sur la page de connexion pour recevoir un e-mail de réinitialisation
  • Compte verrouillé — Après plusieurs tentatives échouées, votre compte peut être temporairement verrouillé. Attendez 15 minutes et réessayez
  • Code à deux facteurs non fonctionnel — Assurez-vous que l'horloge de votre appareil est synchronisée. Les codes d'authentification sont sensibles au temps
